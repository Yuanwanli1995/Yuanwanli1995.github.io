9.1 什么是dockerfile

Dockerfile 是一个用来构建镜像的文本文件，文本内容包含了一条条构建镜像所需的指令和说明。

9.2 dockerfile构建镜像的流程

  1. 编写一个dockerfile文本
  2. docker bulild构建一个镜像
  3. docker run 运行镜像
